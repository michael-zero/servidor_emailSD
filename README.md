# Funcionalidades Implementadas 
1. Enviar Email 
2. Deletar Email
3. Responder Email 
4. Encaminhar Email
5. Listagem de Emails
6. Visualização da Mensagem

# Instalar dependências 
Abra o Vscode ou com o terminal execute o comando 

`npm install`

# Rodar o servidor 
`npm run server`

# Rodar frontend
`npm start`
